# C++ Fetch (cppfetch)

## Introduction
This is my first project made to progress with C++, I made this when I was 14 to start some C++ projects to learn more about it.

## What works:
- Hardware detection such as RAM usage, CPU, GPUs
- Shell detection
- OS detection
- ASCII icon
